## 🔍 **stress_v ＝「銘柄が今どれだけ歪んでいるか」を表す圧力指標**

📌 特徴

| 項目           | 内容                                 |
| ------------ | ---------------------------------- |
| 数値範囲         | **0.0 ～ 1.0（基本）**                  |
| 理想           | 0.3〜0.6程度                          |
| 高すぎ（0.7〜1.0） | 価格が「行き過ぎている」、異常圧力（典型的には高リスク）       |
| 低すぎ（0.0〜0.2） | 動きが停滞 or 相場無関心（＝勝負しにくい）            |
| 計算対象         | 銘柄単体（相対評価ではなく「現在の地合いとの乖離」）         |
| 使用目的         | **「今日その銘柄で仕掛けても良いか」の判断に使う抑制/警告指標** |

---

## 🧠 もっと直感的に言うと…

📌 *「ボールが伸び切ったゴムを更に引っ張るような状態」*
📌 *「過去の統計分布から大きく乖離している状態」*

このとき **“その方向に更に賭けるのは危険”** という判断材料になります。

---

## 📐 stress_v の内部構成（概念的）

厳密にはコードでこういう式ではありませんが、要素としては以下のような **“過熱／圧迫の複合評価”** です：

```
stress_v ≈ 標準化( |price − vwap20| / ATR ) 
        + 標準化(RSIの偏差) 
        + 標準化(出来高の歪み) 
        + ボラティリティ状態 (VIXやセクター指数を反映)
```

条件に応じて重みがかかり、**最終的に 0.0〜1.0 に正規化**されます。

## 📊 よくあるケースと解釈

| stress_v      | 状態             | トレード判断           |
| ------------- | -------------- | ---------------- |
| **0.15**      | 市場が無関心、出来高薄    | トレードしても動かない可能性   |
| **0.30–0.50** | 健全に動いている       | ⚪ “今日トレード対象にできる” |
| **0.60–0.70** | やや過熱気味         | △ “慎重に。トランシェ小で”  |
| **0.75–0.85** | トレンド過熱、マネー流入過剰 | ❌ “追いかけるのは危険”    |
| **0.90+**     | 異常。ショートカバーなど   | 原則ノートレ。          |
| 売り狙いは慎重に      |                |                  |

---

## 🧩 stress_v と「signal」「shk_stat」との違い

| 指標           | 役割                | 判断対象          |
| ------------ | ----------------- | ------------- |
| **stress_v** | 銘柄単体の圧力・乖離        | “今この瞬間の歪み”    |
| **signal**   | 売買方向（BUY/HOLD/TP） | “統計的には買いor利確” |
| **shk_stat** | ショック状態（CRASH等）    | “環境が異常かどうか”   |

👉 *stress_v は強くても **BUY signal が出ている場合は「伸びる可能性もある」***
👉 *ただし **shock時（CRASH/SHEDDING）や stress_v>0.75 は新規エントリーは避ける*** のが基本戦略

## 🔎 最適 threshold 案（実戦例）

```text
stress_v <= 0.60 → 通常通り検討可
0.60 < stress_v < 0.75 → 小ロット or トランシェ1のみ
stress_v >= 0.75 → 新規エントリー見送り
```
